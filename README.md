# 國小寫作 AI 備課指令系統

依據**台灣108課綱**國語文領域學習表現與學習內容設計的國小語文教師 AI 備課工具。

## 適用對象

- 國小一至六年級語文教師（第一～第三學習階段）
- 包含非語教專長的全科教師
- 適合搭配 ChatGPT 或 Claude 使用

## 功能模組

| 模組 | 名稱 | 說明 |
|------|------|------|
| A | 學情診斷與課程定位 | 診斷學生能力層次、找出能力斷點與教學起點 |
| B | 單元寫作教學設計 | 依年段、文體、節數生成完整教學計畫 |
| C | 課堂任務與鷹架建構 | 形式鷹架、內容鷹架、分層差異化任務 |
| D | 評量規準與回饋設計 | 評量規準、AI輔助批閱、學生自評互評工具 |

## 使用方式

1. 開啟網頁，選擇對應模組
2. 填寫基本教學資訊（年級、文體、班級描述等）
3. 點擊「生成指令」按鈕
4. 複製生成的指令，貼入 ChatGPT 或 Claude 使用

## 課綱依據

- 108課綱學習表現：6-I 至 6-III（國小階段）
- 學習內容：文字篇章（Ab/Ac/Ad）、文本表述（Ba/Bb/Bc/Bd/Be）

## 版本說明

- **目前版本**：國小版（1–6年級）
- **開發中**：國中版（7–9年級）、國小國中銜接診斷模組

## 技術說明

純前端靜態網頁，單一 `index.html` 檔案，無需伺服器或資料庫。
所有資料在本地瀏覽器處理，不上傳任何教師或學生資料。
